# 算数运算符和表达式
---
## 一、算数运算符

1. 加法运算符（+）

    • 用于两个操作数的相加。
    例如：a + b 结果为 a 和 b 的和。
    
2.减法运算符（-）
    
    • 用于两个操作数的相减。
        例如：a - b 结果为 a 减去 b 的差。
    
3. 乘法运算符（*）
    
    • 用于两个操作数的相乘。
        例如：a * b 结果为 a 和 b 的积。
    
4.除法运算符（/）
    
    • 用于两个操作数的相除。
        整数相除时，结果为商（小数部分会被截断）；浮点数相除则保留小数。
        例如：a / b 结果为 a 除以 b 的商。
        
        int a = 10, b = 3;
int quotient = a / b;    // quotient 等于 3，整数除法，舍去小数部分
        
        对于浮点数：
        float x = 10.0, y = 3.0;
float result = x / y;    // result 等于 3.3333
    
    1. 取模运算符（%）
    
        • 用于计算两个操作数相除后的余数。只能用于整数。
        例如：a % b 结果为 a 除以 b 的余数。
        
        int a = 10, b = 3;
int remainder = a % b;    // remainder 等于 1
        
    6.递增/减运算符（++）
    
        • 用于将操作数的值 加 / 减  1。
        
        • 有前缀递增/减（++a）和后缀递 增 / 减（a++）两种形式：
            ○ 前缀递 增 / 减 ：先 增加 / 减少 变量的值，然后再使用它。
            ○ 后缀递 增 / 减 ：先使用变量的值，然后再 增加 / 减少 它。
            
            int a = 5;
            int b = --a;    // 先递减 a，b = 4，a = 4
            int c = a--;    // 先使用 a 的值，c = 4，a 递减到 3
